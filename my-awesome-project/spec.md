# My Awesome Project

## Overview
A React SPA with a single green button that displays a "My divine website!" message with a Close button.

## Features
1. Green button on the main page
2. Clicking the green button shows a message "My divine website!" and a "Close" button
3. Clicking the "Close" button hides the message

## Technical decisions
- Language: TypeScript
- Framework: React 19 + Vite
- Test runner: Vitest + React Testing Library
- Styling: CSS (inline or module)
